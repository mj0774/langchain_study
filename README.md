# LangChain & LangGraph Study


## 프로젝트 개요
이 레포는 **LangChain**과 **LangGraph**의 내부 동작 구조를 학습하기 위한 실습 코드입니다.  
대부분의 소스는 **LangChain docs**에 기반하고 있으며 **LangChain 1.0.3** 기준의 문법으로 작성되었습니다.

## 개발 환경
#### python 3.11
#### langchain 1.0.3
#### langchain-openai 1.0.1
#### langgraph 1.0.2
#### langsmith 0.4.40
